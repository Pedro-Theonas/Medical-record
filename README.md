Medical Form (Printable A4)

This project is a simple web-based medical form designed to be filled out and printed in A4 format. It provides a clean layout that mimics a real paper form, making it ideal for printing or saving as a PDF.

Features
Fill out basic medical form information
Clean A4 layout optimized for printing
Upload a custom logo from your computer
Automatic image resizing to fit the layout
Print or save as PDF with a single click
How It Works

The page displays a structured medical form where users can enter personal and clinical information. The layout is fixed to A4 proportions to ensure everything prints correctly.

One of the key features is the ability to upload a logo image. No matter the image size, it is automatically adjusted to fit داخل the designated space without affecting the rest of the layout.

When the print button is pressed, the interface removes unnecessary elements (like buttons and background), leaving only the form for a clean print or PDF export.

Technologies Used
HTML – Structure of the form
CSS – Styling and print formatting
JavaScript – Interactivity (logo upload and printing)
Usage
Open the index.html file in your browser
Fill out the form fields
Click "Choose Logo" to upload an image
Click "Print" to print or save as PDF
